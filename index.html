Measuring current using a coil of Fiber-Optic cable
Electrical Engineering Laboratory Design
2021

Investigation on designing an optical current measurement system using the Faraday effect. The investigation details how it is possible to measure current non-intrusively using such a system. It's found that polarization rotation of a beam can be observed and used to correlate this rotation with the current producing a magnetic field. It is shown using DC measurements the polarization rotation is clearly visible when birefringed by a high magnetic field. The system showed a high error for low currents and a reliable accuracy for currents ≈≥ 200A/13. It is suggested in the investigation to perform the investigation using a lock-in amplifier for phase sensitive detection to improve the investigation and to repeatedly do the investigation for currents ≈≥ 200A/13 to determine the Verdet constant of the fiber optic coil reliably.

Worked in a team to collaboratively develop and apply SDLC for a web application for splitting bills in a shared household. Indexing XML Hansards
Data Intensive Computing in Data Science
2020

Introduction to the principles of the MapReduce framework for processing big data. The main features in such a framework are the reliable and fault-tolerant processing techniques employed. While not all problems are solvable by the MapReduce techniques, a large number of decomposable problems can be divided up to be solved by the MapReduce technique. The implementation was a C indexing program that pre-processes and represents the extensive data of 20th century XML Hansards(Paliamentary Debates). The designed program used the commercial software tool Pheonix++'s MapReduce for processing data and reducing the rate at which the data can be timeously indexed and searched using multi-threading API's OpenMP and MPI.
